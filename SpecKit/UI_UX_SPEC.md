\# UI/UX \& Interaction Specification - HeuristicLogix



\## 1. Planning Dashboard (The Expert Workspace)

\- \*\*Layout:\*\* MudBlazor-based dashboard.

\- \*\*Interaction:\*\* Drag \& Drop interface using `MudDropContainer`.

\- \*\*Workflow:\*\* Dragging a Conduce from the "Pending" column to a "Truck Bucket" updates the record and triggers a route recalculation.



\## 2. Smart Data Entry

\- \*\*Component:\*\* Google Places Autocomplete integration.

\- \*\*Visual Validation:\*\* Real-time map preview next to the address field to confirm the delivery point location.
